🔘 Button-Label Test-Seite

Diese Seite testet die automatische Erkennung von Buttons ohne zugängliche Namen.

❌ PROBLEME - Diese Buttons sollten erkannt werden:

1. Icon-Button nur mit Symbol (Unicode)

2. Button mit nur CSS-Icon (kein Text)

3. Button mit nur Bild (ohne Alt-Text)

4. Leerer Button

5. Button mit nur Leerzeichen

✅ KORREKT - Diese sollten NICHT als Problem erkannt werden:

6. Button mit sichtbarem Text

7. Icon-Button mit ARIA-Label

8. Button mit aria-labelledby

Dokument speichern

9. Button mit title-Attribut

10. Button mit Bild UND Alt-Text

🧪 Test-Anleitung:

  1. Rechtsklick auf die Buttons im PROBLEME-Bereich
  2. Wähle "Note zu diesem Element hinzufügen"
  3. Prüfe, ob automatisch "Button ohne zugänglichen Namen" oder ähnliche Probleme erkannt werden
  4. Teste auch die Buttons im KORREKT-Bereich - hier sollten KEINE Button-Label-Probleme erkannt werden

Erwartete BITV-Referenz: BITV 2.4.6 - Überschriften und Beschriftungen
